How to run this project :

1. Clone this repo via terminal / download zip file

``` 
$ git clone git@github.com:Dhimassudaryanta/rn-mobile-app-test.git

$ cd rn-mobile-app-test

$ npm install
```

2. To start the app, you can run:

```
$ npm start
```
